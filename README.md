## Projects

### Housing Repairs Management System – UAT Bug Report

During User Acceptance Testing (UAT) of a housing repairs management system, I identified a workflow issue when creating repair variations.

The issue occurs when navigating away from the variation page, which returns the user to the previous screen instead of keeping them within the repair variation workflow.

This project includes:
- Bug summary
- Steps to reproduce
- Expected vs actual results
- Context of the issue
